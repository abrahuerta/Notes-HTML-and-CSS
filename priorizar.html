Priorizar Estilos

A medida de que vayamos creando estructuras complejas en html, frecuentemente ocuparemos asignar varios estilos a un mismo elemento, asi que es importante saber como priorizar los estilos de los elementos.

Ejemplo: Si yo le asigno dos clases al elemento h2 estas deberan ir en el mismo metodo class solo separadas por un espacio, estas dos clases tendran conflicto ya que no pueden aplicarse 2 estilos a la vez, entonces alguna de las 2 tendra una prioridad. Sorprendentemente en HTML no importa el orden de las clases, asi que si colocas una clase primero y luego la otra no se priorizar√° por orden. Es en CSS donde la prioridad hace efecto, ya que CSS toma la ultima clase que se encuentre en el archivo CSS.

Si los estilos son distintos en el selector, (un estilo con selector de tipo y otro estilo con selector de clase) CSS le dara prioridad al selector mas especifico (en este caso el de clase) sin importar su orden en la hoja de estilo CSS
