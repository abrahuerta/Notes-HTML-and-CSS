Imagenes y Atributos

Las imagenes es una parte central para la web, para agregar una imagen ocupamos la etiqueta "img" recuerda que no es necesaria la etiqueta de cierre. En este caso ocuparemos una imagen de internet (no descargada en nuestros ficheros), Ocuparemos el Atributo src = "" (source) donde dentro de las comillas pondremos nuestra URL donde esta alojada la imagen que queremos como src es un atributo de la etriqueta "img" este debe estar dentro de la etiqueta de apertura

Un atriuto importante que deberia tener una imagen en html es el atriuto alt = "" ya que es una descripcion alternativa para la imagen que se va apreesentar si la imagen no se descarga correctamente o si el usuario esta usando un lector de pantalla, ese texto se va a leer.

Para cargar una imagen desde el dispositivo en el src = "" pondremos la ruta del archivo dentro de nuestro dispositivo, si esta en la misma carpeta donde esta el programa podemos poner unicamente el nombre del archivo.
