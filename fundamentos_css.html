Fundamentos CSS


CSS significa por sus siglas Cascade Style Sheets (Hojas de estilo en cascada)

En Html definimos la estructura y en css el estilo de esa estructura

CSS es un lenguaje basado en reglas esas reglas determinan el estilo que se aplicara a un elemento o a un grupo de elemenotos


Opciones para definir reglas en CSS

Estilos en linea

El estilo que vamos a asginarle a un elemento se añade directamente a la etiqueta de apertura de ese elemento en el archivo html, se coloca el attributo style = "" y entre las comillas ("") se coloca la propiedad que se va a personalizar, en este caso es el color del texto.

Etiquetas de apertura y cierre "style"

Cuando usamos el elemento "style" añadimos ese elemento en el head de nuestro html y dentro de ese elemento describimos el estilo que queremos asignarle a los elementos de la página; basicamente agrupamos las reglas bajo un mismo elemento en lugar de hacerlo elemento por elemento en su etiqueta correspondiente y mejor agrupamos las reglas en un mismo bloque

Ejemplo:

head
style
h2 {color = "blue";
}
/style
/head

Donde h2 y los demas elementos contenidos en las etiquetas "style" son selectores css, esto nos permite que todos los elementos que tengan esa etiqueta se ele apliquen en este caso el color azul


Desde un Archivo .css

En esta opción creamos un archivo css para definir el estilo e la página, este archivop contendra todas las reglas que se aplicaran al html, basicamente tendremos un archivo html y un css. Para vincular el css con el html se hace con el elemento link en su atributo href se coloca la ruta del archivo css.

Ademas del elemento href también se especifica el atributo rel = "" este atriuto denomina la relación del documento vinculado al actual en este caso seria de tipo "stylesheet"
