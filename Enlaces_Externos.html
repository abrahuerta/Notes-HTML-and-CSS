Enlaces Externos

En HTML un enlace se crea con la etiqueta "a" (anchor) podemos convertir una parte de nuestro texto en un hiperviculo por ejemplo: Haga click aqui para ver imagenes de perritos bonitos.

El url del hiperviculo se tiene que poner como valor del atributo href = "" dentro de la etiqueta de apertura "a" y asi el contenido de las etiquetas "a" se mostrara de color azul indicando que es un hipervinculo

Para que el hipervinculo nos abra el url en otra pestaña debemos ocupar el atributo target = "", este es un atriuto que nos va permitir especifiar donde queremos que se abra el enlace, su valores pueden ser:

------------------------------------------

target = "_blank" ----> Abrira el hipervinculo en una pestaña nueva

target = "_self" -----> Abre el documento vinculado en la mismo pestaña en el que se hizo clic (esto es predeterminado)

target = "_parent" ---> Abre el documento vinculado en la pestaña principal.

target = "_top" ------> Abre el documento vinculado en el cuerpo completo de la ventana

target = "framename" -> Abre el documento vinculado en el iframe nombrado

------------------------------------------
------------------------------------------

El atriuto rel = "" El atributo especifica la relación entre el documento actual y el documento vinculado ya que los motores de busqueda pueden usar este atributo para obtener información sobre un enlace.

El atributo rel = "" nos permite prevenir un ataque llamado tabnabbing, este ataque hace que la pestaña de origen se reemplace para que el usuario ingrese sus datos y eso es una brecha de seguridad.

------------------------------------------

rel = "alternate" ---> Proporciona un enlace a una representación alternativa del documento (es decir, página impresa, traducida o reflejada)

rel = "author" ---> Proporciona un enlace al autor del documento.

rel = "bookmark" ---> URL permanente utilizada para marcar

rel = "external" ---> Indica que el documento al que se hace referencia no forma parte del mismo sitio que el documento actual

rel = "help" ---> Proporciona un enlace a un documento de ayuda.

rel = "license" ---> Proporciona un enlace a la información de licencia del documento.

rel = "next" ---> Proporciona un enlace al siguiente documento de la serie.

rel = "nofollow" ---> Enlaces a un documento no respaldado, como un enlace pagado.("nofollow" es utilizado por Google para especificar que la araña de búsqueda de Google no debe seguir ese enlace)

rel = "noopener" ---> Requiere que cualquier contexto de navegación creado siguiendo el hipervínculo no debe tener un contexto de navegación de apertura

rel = "noreferrer"---> Hace que el referente sea desconocido. No se incluirá ningún encabezado de referencia cuando el usuario haga clic en el hipervínculo

rel = "prev" ---> El documento anterior en una selección

rel = "search" ---> Enlaces a una herramienta de búsqueda del documento.

rel = "tag" ---> Una etiqueta (palabra clave) para el documento actual
