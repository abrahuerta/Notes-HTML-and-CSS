Paddign

El padding en html es la distancia que hay entre el borde de un elemento y su contenido. El siguiente div contiene una imagen, y a continuación vamos a modificar los bordes del elemento img con el contenido del elemento, sus unidades generalmente son en "px" y su sintaxis es asi:

img{

padding: 15px;

}

Si el padding se aplica a un elemento, este sera la separación del borde del elemento y el contenido

Padding para cada lado del elemeto

Para personalizar el padding de un elemento, se cuentan con las reglas css:

  • padding: 15px; (se aplica uniformemente a cada lado del elemento)
  • padding: auto;
  • padding-left: ;
  • padding-top: ;
  • padding-bottom: ;
  • padding-right: ;
  • padding-inline-end: ;
  • padding-inline-start: ;
  • padding-block: ;
  • padding-block-end: ;
  • padding-block-start: ;
  • padding-inline: ;

Asi como el border, la regla padding soporta 4 distintos tamaños, esto para cada uno de los lados del elemento, en el orden de las manecillas del reloj (top,right,bottom,left), y se puede escribir en una sola linea para no escribir las en distinas lineas.

Lo que antes se escribia asi:

img{

padding-top: 10px;

padding-left: 23px;

padding-bottom: 15px;

padding-right: 5px;

}

Se escribe asi: para ahorrar espacio

img{

padding: 10px 23px 15px 5px;

}
