Formularios (input)

type = "Text"

Para crear formularios en html ocupamos la etiqueta "form" (formulario), se utiliza etiqueta de apertura y de cierre, un elemento que se puede tener dentro de un formulario es un campo "input" este elemento lo usamos para obtener un valor o una respuesta del usuario, si queremos que el usuario escriba una respuesta o escriba un valor se puede asignar el tipo de input como "text" que es un atributo del elemento input (osea va dentro de la etiqueta de apertura de input) se declara de la sig forma type = "text"

Se puede indicar dentro de la casilla un texto cualquiera, pero por lo general se ponen una pista al usuario para que sepa que tipo de texto puede ingresar y se hace con el atributo placeholder (marcador de posicion) y se especificara dentro de la etiqueta de apertura del input en cuanto el usuario comienze a ingresar texto este texto de pista se borrara automaticamente.

Si requieres que un campo de un formulario sea obligatorio (por ejemplo correo obligatorio, etc) lo especificamos en el atriubuto required perteneciente al elemento inputsi enviamos el formulario sin contenido nos saldra un mensaje por defecto indicando que el campo esta vacio y requiere un valor

Para almacenar la respuesta o el valor introducido por el usuario se tiene que especificar dentro de la etiqueta de apertura del form el atributo action = "" donde indicara el archivo en el servidor que se encargara de la gestion de los datos del programa. Si el archivo no existe aparecera un error al enviar los datos.

Los datos se envian con el elemento button este estara anidado dentro del form a la misma altura del input. En el atributo type ="" indicaremos que tipo de boton ocuparemos (en este caso submit ya que enviaremos datos).

type = "radio"

Los formularios tipo radio son parecidos a los tipicos examenes de opción multiple, las cuales tienes que marcar la casilla de la opcioón a elegir esto se hace poniendo dentro de la etiqueta de input el type = "radio"

Para que al hacer click al texto tambien se marque la casilla correspondiente existe el elemento label, este se va a superponer a nuestro input tipo radio para que todo lo que este dentro del label se pueda seleccionar como si fuera la casilla

Para relacionar el contenido del input con el label se utiliza un Id para el input y especificarlo en el atributo "for" del label (en la etiqueta de apertura)

Si marcamos las 2 casillas no hay ningun problema porque aun no hay ninguna restricción acerca de seleccionar las casillas, si queremos que el usuario pueda seleccionar unicamente una casilla los inputs de las opciones deben permanecer a un grupo de botones, esto se implementa de la siguiente forma: Dentro de cada label en su elemento "input", en la etiqueta de apertura se debe especificar el atributo name = "" con ese atributo todos los los inputs que lleven ese nombre formaran parte de un grupo, y solo habra una opción correcta por cada grupo, entonces por si se selecciona 2 opciones una de estas se desmarcara automaticamente.
