Configurar Idioma de la pagina html

Dentro de la etiqueta de apertura HTML se agregara un atributo para especificar el idioma "lang = "es" ", este atributo especifica que todos los elementos dentro de HTML estaran en ese idioma especificado (es), si se pone dentro del elemento Head, entonces todos los elementos creados dentro de head estaran en ese idioma (es) y asi jerarquicamente.
