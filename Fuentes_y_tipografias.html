Fuentes y tipografias

Para personalizar el tamaño de la letra se ocupa la propiedad de CSS "font-size" (tamaño de fuente), hay distintas unidades de tamaño que se pueden utilizar una de ellas y la mas utilizada es "px" (pixeles)

h2 {

font-size: 14px;

}


Para personalizar el tipo de fuente se utiliza la propiedad "font-family"(familia de fuente), si el nombre de la tipografia es de una sola palabra el valor no se encierra con comillas, si el nombre de la tipografia contiene mas de una palabra, se utilizan las comillas

La propiedad font-family definine una lista de fuentes con un orden de prioridad para usar en un elemento seleccionado, podemos especificar varias fuentes, en orden de prioridad, esto se puede por si el browser o navegador no reconoce la fuente principal, entonces se podra ocupar otra como un respaldo, a diferencia de la mayoria de las propiedades CSS, los valores se separan con "," para indicar que son valores alternativos.

h2 {

font-family: verdana, arial;

or

font-famliy: "arial cuerpo"

}

Google Fonts

Otra opción para personalizar las fuentes es Google Fonts, google se encargo de crear una colección de fuentes que podemos importar a nuestra pagina y mostrarlas en nuestros elementos, aqui puedes ver distintas fuentes

Para agregarlo a nuestra página, seleccionamos la fuente que más nos haya gustado, y nos da 2 metodos de importación (link e import), si seleccionamos link, nos muestra un pedazo de codigo para que lo copiemos en nuesta cabecera del html (head), básicamente es un elemento "link" que importa el origen de la fuente. Y despues, agregamos el cogigo css correspondiente en nuestro archivo .css
