Cabecera (Head)


El elemento head va antes del "body", este contiene la Meta-Data y los elementos que van detras de escenas osea los que no vemos pero que son escenciales para que la página funcione correctamente, por ejemplo si creamos un archivo css o javascript y necesitamos incluirlo a la página aqui se incluiria el enlace, ademas también se puede personalizar el titulo de la página con el elemento "title" es importante tener un buen titulo porque esto es un factor importante para la optimización de motor de busqueda SEO asi que se recomienda un buen titulo.

Pie de página


Otro elemento importante es el pie de pagina, esto se agrega al final de la esctructura de nuestra pagina (si se tiene elemento main, pues al final del main) sus etiquetas es "footer"(pie de página) dentro de este se puede incluir un parrafo

La etiqueta "small" hara el texto contenido mas pequeño
