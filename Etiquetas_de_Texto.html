Etiquetas de formatos para Textos

Si queremos que parte de nuestro texto tenga formatos (negrita, cursiva, etc) hay etiquetas especificas para cada caso

  • Para las letras en negritas tenemos la etiqueta "strong" (fuerte) Donde el texto dentro de las etiquetas se pone en negritas
  • Para las letras en cursiva se ocupa la etiqueta "em" Donde el texto dentro de las etiquetas se pone en cursivas
  • Para poner un texto tachado se ocupa la etiqeuta "s" (strikethrough) Donde el texto dentro de las etiquetas se muestra tachado
  • Para agregar lineas horizontales las cuales nos ayudan a dividir secciones de la pagia se puede agregar con la etiqueta "hr" (horizontal rule)

El aspecto de hr se puede cambiar con css pero por defecto es la que aparece arriba y solo requiere la etiqueta de apertura no la de cierre :D
