Selectores CSS

Los selectores ya los usamos en el estilo de "elemento style" y el estilo "crear un archivo .css", tiene esta forma

h2 {

color: red;

}

En la parte donde esta h2, indicamos el tipo de elemento al que se le va a aplicar el style (en este caso es color), puede haber muchas combinaciones de propiedades a modificar dentro de las llaves, el style que especifiquemos dentro de los corchetes aplicara para el elemento especificado

Es escencial indicar a que elemento se le aplicara el style ya que el elemento le indica al browser a quienes se les aplicara los estilos

Hay muchos tipos de selectores CSS, los mas importantes son:

  • Selector de Tipo
  • Selector de Clase
  • Selector de ID
  • Selector Universal
  • Selector de Atributo

Selector de Tipo

El selector de tipo es el que hemos estado manejando este tiempo, selecciona todos los elementos que coinciden con el nombre del elemento especificado.

h2 {

color: red;

}

Selector de clase

Este selector selecciona todos los elemento que tengan el atributo class especificado.

Por ejempo: Todos los elementos con class = "spacious"

.spacious {

color: red;

}

Selector de ID

Este selector selecciona un elemento basándose en el valor ID de su atributo. Solo puede haber un elemento con un determinado ID dentro de un documento.

Por ejemplo: Todos los elementos que su id sea id = "prueba"

}

#prueba {

color: red;

}

Selector Universal

El selector Universal selecciona todos los elementos. Opcionalmente, puede estar restringido a un espacio de nombre específico o a todos los espacios de nombres.

Por ejemplo: Aplicar a todos los elementos el texto color verde

* {

color: green;

}
