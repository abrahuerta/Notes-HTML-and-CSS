Imagenes

width

Se puede modificar el tamaño de las imagenes, para presentarlas como nosotros queramos con la propiedad "width"(ancho), el ancho puede tener varias unidades en este caso utilizamos "px", si se quiere modificar solo algunas imagenes, siempre esta la opción de selectores de clases.


Bordes

Los bordes se pueden personalizar con 3 propiedades importantes (hay muchas mas propiedades pero las principales son estas 3)

  • border-width: Ancho del borde, sus unidades principales es en "px", entre mayor sea el valor, mas ancho es el borde
  • border-color: El color del borde
  • border-style: Hay muchos tipos de bordes, el basico es el solid, que es una linea continua al rededor del elemento, puedes ver mas en este link

border-style soporta hasta 4 estilos diferentes, para los 4 lados de una imagen, estos deben ir separados por un espacio, no por comas.

Redondear

La propiedad para redondear los bordes se llama "border-radius"(radio del borde), entre mayor sean los pixeles, mayor sera el efecto de redondeo este valor tambien se puede representar por porcentajes
