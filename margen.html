Margen

El margen es la distancia del borde de un elemento y el borde de otros elementos, basicamente es como separar los elementos, a diferencia del padding que es separar el borde con el contenido del elemento. Sus unidades generalmente es en px

Su sintaxis es la sifguiente:

img{

margin: ;

}

Ajuste de margen

Las mismas propiedades de ajuste de padding se pueden aplicar al margen para modificarlo al gusto ya sea de esta forma:

img{

margin-top: 10px;

margin-left: 23px;

margin-bottom: 15px;

margin-right: 5px;

}


o de esta forma:

img{

margin: 10px 23px 15px 5px;

}

Diferencia entre Margin y Padding

padding es la distancia entre el contenido del elemento y el borde por otro lado el margin es la distancia del elemento con otros elementos, el padding se puede considerar como distancia interna y el margin como distancia externa
