Elementos

Un elemento es un componente basico de una archiuvo html, un elemento es como un ladrillo que se utiliza para definir la estreuctura de una web en un archivo html El archivo es donde se va a escribir nuestro codigo, los elementos seran los componentes basicos del codigo y esos elementos se combinan para crear la estrucutira de la pagina web.

Etiquetas

Los elementos y las etiquetas estan muy relacionadas ya que las etiquetas nos permiten definir los elemntos que usamos en un archivo html, basicamente las etiquetas son las formas de esceribir esos elemenstos en un archivo, nosotros escribimos las etiqueas en nuestro archivo html y esas etiquetas crean los elementos de la pagina web las etiquetas especifican el tipo de elemento que se va a crear en esa parte especifica de la pagina web "TAG"

La etiqueta de apertura empiza por "<>" y termina con la etiqueta de cierre '' En general la estrucuta de un elemento es Esta "" lo que este dentro de las etiquetas de apertura y las ertiquetas de cierre es el contenido del elemento

Etiquetas self-closign

Algunos elementos de html no necesitan etiquetas de cierre por ejemplo la etiqueta img "" ya que con la etiqueta de apertura es suficiente y automaticamente se va a cerrar, esta etiqueta nos permite insertar una imagen en nuestra pagina web
