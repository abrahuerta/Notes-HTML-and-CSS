Elemento style

Para reglas de css con el elemento style se escribe dentro del head, si no se describe especificamente a que elemento va dirigido el style, este se aplicara a todos los elementos por ejemplo, tenemos 4 h2 y en el selector solo ponemos el style para h2, este aplicara a todos los h2 por eso hay que tener cuidado al definir las reglas css en elementos style
